# Módulos ECUABYTE para Odoo 17

Este repositorio contiene módulos personalizados desarrollados por ECUABYTE para Odoo 17 Enterprise.

## Descripción General

El objetivo de este repositorio es centralizar módulos reutilizables enfocados en la automatización de procesos empresariales, mejoras en CRM, gestión de oportunidades de contratación pública, integraciones con inteligencia artificial y optimización de la experiencia de usuario.

## Módulos Disponibles

### Módulo SERCOP

Automatiza la obtención y gestión de oportunidades provenientes del Sistema Oficial de Contratación Pública del Ecuador (SERCOP), permitiendo a las organizaciones optimizar el seguimiento de oportunidades y los procesos comerciales directamente desde el CRM de Odoo.

### SERCOP AI

Extensión basada en inteligencia artificial diseñada para analizar oportunidades de contratación pública, generar información relevante y apoyar la toma de decisiones estratégicas.

### Tema ECUABYTE

Tema visual personalizado que mejora la interfaz de usuario de Odoo, proporcionando una experiencia moderna, profesional y alineada con la identidad corporativa.

## Tecnologías Utilizadas

* Odoo 17 Enterprise
* Python
* PostgreSQL
* JavaScript
* SCSS
